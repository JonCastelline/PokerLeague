# PokerLeague

Initial ideas:

User types:
	Super Admin: Access to all functionality and can add and remove Admins.
	Admin: Access to all functionality within a league.
	User: Access to all funcitonality unless specified to Admin.

Main Menu:
	Logo (part of CSS or just first page?)
	Hamburger Menu (part of CSS top right):
		Log in/out
		Profile
		Leagues
		Admin (Admin only)
	Buttons:
		Tournament Game
		Standings
		Schedule
		Rules
		
Logo: Updates based on league currently on.  Has app default logo until in a league that has uploaded a logo.

Log in/out:
	If not logged in:
		Username
		Password
		Create username button
	If logged in:
		Log out
		
Profile:
	
	
Create username button:  Takes to new screen.
	Enter e-mail.
	Verify e-mail with code.
	Enter code.
	Enter username.
	Enter new password.
	Confirm password.
	
Tournament Game: This is meant to replace the functionality of the program currently used.  All functionality syncs across devices.
	Timer
	Blinds
	Next level blinds
	Buttons:
		Start/stop: Starts/stops the timer.
		Eliminate (Admin only)
